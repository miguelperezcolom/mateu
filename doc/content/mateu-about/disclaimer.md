# Current status

Mateu is currently undergoing a significant refactor (version 3).

The core architecture is stable and the API is defined, but some features from version 2 are still being migrated. **Version 3 is expected to reach production-ready state in 2026.**

If you're evaluating Mateu for a project, the best way to assess its current capabilities is to:

1. Try the [live demos](../see-it-in-action) — they run on v3
2. Watch the [v3 preview video](https://youtu.be/L7bKeijIqjY)
3. Check the [GitHub repo](https://github.com/miguelperezcolom/mateu) for current activity

---

## What works now (v3)

- Core UI rendering (forms, lists, actions, navigation)
- Stateless interaction model
- Multiple design system renderers (Vaadin, Red Hat, SAPUI5, Redwood)
- UI federation via `RemoteMenu`
- Web component embedding
- Spring Boot MVC and Webflux integration

## In progress

- Full documentation for all features
- Kotlin, C#, and Python support
- AI integration (MCP server, LLM-friendly code generation)
- Some advanced components from v2
