# UI model in Mateu

Mateu UI emerges from:

- inference
- layout
- regions
- stereotypes
- style

---

## Style

`@Style` provides fine control.

It can be used for:

- individual fields
- layout tweaks
- page-level constraints

---

## Mental model

> inferred → structured → placed → refined

Style is the final refinement layer.
