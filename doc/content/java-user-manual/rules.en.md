# Rules

Mateu lets you define dynamic client-side behavior declaratively with `@Rule`.

Rules are evaluated in the browser and can modify:

- field attributes
- values
- styles
- CSS classes
- actions

## Mental model

Rules define how the browser UI changes dynamically.
